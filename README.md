# Oven_management_system
Oven management system that is a system that provide you to access you oven remotely. We can get the live data of oven on our devices. We also get direct alert messages on out registered devices in case of any mishappening. Also helps in reducing risk.

Monitoring of: 
• Temperature Value 
• Job Schedule 
• Time based graph of last few hours of temperature value 

Control Of: 
• Job Schedule 
• Alert Levels 
• Upload Interval 
• Device Time 
• Device Restart 
• Cloud Configurations 
• User Authentication 
An administrator can schedule a job using Android Application and, if need be, can stop a job 
schedule in real time by sending SMS command.
